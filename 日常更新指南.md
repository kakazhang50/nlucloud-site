# NLU Cloud 日常更新指南

## 发一篇新 Essay

1. 打开 `src/content/essays/[slug].md`
2. 填写正文，将 frontmatter 中 `status: planned` 改为 `status: published`
3. 设置 `pubDate`、`category`、`sourcePaper`
4. 本地预览：`npm run dev` → http://localhost:4323
5. Push 到 GitHub → Cloudflare Pages 自动部署

## 从论文规划批量占位

- 清单：`src/data/essayManifest.json`（52 篇创刊库存）
- 生成占位文件：`npm run gen:essays`（已存在的文件不会覆盖）

## 新论文上线

1. 新建 `src/content/researchPapers/[slug].md`
2. 可选：新建 Framework 页
3. 在 manifest 中追加 Essay 条目，或手写 essay 文件
4. `npm run build` 验证

## 同一论文多篇 Essay

- `sourcePaper` 相同，`category` 可不同（如 ALT 同时进 `ai-life` 与 `ai-society`）
- Research 页会自动聚合同 `sourcePaper` 的已发布 Essay

## 关键路径

| 改什么 | 文件 |
|--------|------|
| 全站文案 | `src/data/siteConfig.json` |
| 六栏定义 | `src/data/categories.ts` |
| Essay 清单 | `src/data/essayManifest.json` |
| GEO 短版 | `public/llms.txt` |
